### 3-2-1 Grid
```css
/* Desktop */
grid-template-columns: repeat(3, 1fr);  /* 3 columns */

/* Tablet */
@media (max-width: 1024px) {
  grid-template-columns: repeat(2, 1fr);  /* 2 columns */
}

/* Mobile */
@media (max-width: 768px) {
  grid-template-columns: 1fr;  /* 1 column */
}
```

}
